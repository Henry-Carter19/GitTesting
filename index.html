Git Testing Command Palette

A collection of common Git commands for testing purposes. Click on the code blocks to select and copy.

Initialize Repository
git init
Add All Files
git add .
Commit Changes
git commit -m "Your message"
Check Status
git status
View Commit History
git log --oneline
Create Branch
git branch feature-branch
Switch Branch
git checkout feature-branch
Merge Branch
git merge feature-branch
Clone Repository
git clone <repository-url>
Push to Remote
git push origin main
Pull from Remote
git pull origin main
Create Tag
git tag v1.0
Show Differences
git diff
Stash Changes
git stash
Apply Stash
git stash pop
